## 环境

1. 下载安装 node

2. 安装 cnpm
```
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
```

3. 下载安装git-bash




## 启动开发

1. 克隆代码
```
$ git clone 代码地址
```

2. 安装依赖
```
$ cnpm install
```

3. 启动开发环境
```
$ npm start
```
## 生产（打包）
```
$ npm run build
```

## 其它
- 查看git配置
```
$ git config -l
```

- SSH测试连接
```
$ ssh -T git@github.com
```
